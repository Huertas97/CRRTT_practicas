Usando la funcin gnp_random_graph(n,p) de NetworkX Implementad una rutina Python L_C_Aleatorio que devuelva valor promedio de los coeficientes C (ndice de clusterizacin promedio) y L (camino caracterstico) de un conjunto de n_graph grafos aleatorios de n nodos y probabilidad p.

def L_C_Aleatorio(n_graph,n,p):
    mu_C = 0
    mu_L = 0
    for i in range(n_graph):
        G = nx.gnp_random_graph(n, p)
        C = nx.average_clustering(G)
        if nx.is_connected(G):
            L = nx.average_shortest_path_length(G)
        else:
            L = n-1
        mu_C += C/n_graph
        mu_L += L/n_graph
        
    return mu_L, mu_C

Valor crítico de $p$, demostración¶

Por un lado, tenemos que $<k> = \frac{2|E|}{|V|}$ y $ |E| = p\frac{|V|(|V|-1)}{2} \;.$ Despejando de las dos ecuaciones, es fcil ver que:

$$<k> \simeq p|V| \;.$$

Para un grafo aleatorio, Erdos y Renyi demostraron que la estructura de clusters de un grafo cambia abruptamente cuando $<k>$ se acerca a $1$. La probabilidad crtica ser por tanto:

$$ k\simeq 1 \rightarrow p_c|V| \simeq 1 \rightarrow p_c \simeq \frac{1}{|V|} \;.$$

Para nuestros grafos, que tienen $|V| = 400$ nodos, deberamos obtener una probabilidad crtica $p_c = \frac{1}{400} = 0.0025$.

  • Tiene sentido que en ambas grficas cambien su tendencia en ese punto, porque a medida que crece la C el camino caracterstico entre nodos disminuye; estn ms conectados, y el ndice de clusterizacin aumenta.

def L_C_Small_World(n_graph,n,k,p):
    mu_C = 0
    mu_L = 0
    for i in range(n_graph):
        G = nx.watts_strogatz_graph(n, k, p)
        C = nx.average_clustering(G)
        if nx.is_connected(G):
            L = nx.average_shortest_path_length(G)
        else:
            L = n-1
        mu_C += C/n_graph
        mu_L += L/n_graph
        
    return mu_L, mu_C

Para un grafo de mundo pequeo se comprueba que el camino caracterstico disminuye bruscamente (cambio de rgimen) para el valor terico:

$$p = \frac{1}{k|V|} \;.$$

En nuestro caso, para $k=8$ y $|V| = 400$, tendramos que la probabilidad crtica

$$ p_c = \frac{1}{8\times 400} = 0.0003 \;.$$

  • Nmero de nodos |V| = 10000
  • Nmero de aristas |E| = 99900 (ver info de G_BA)

Con la ecuacin:

$$|E| = p\frac{|V|(|V|-1)}{2}$$

Podemos despejar $p$, obteniendo:

$$p = \frac{2|E|}{|V|(|V|-1)} = 0.01998$$

  • La red aleatoria tiene una distribucin de grados que sigue una Poisson, como podemos observar claramente en el histograma generado para este tipo de grafos.
  • Sin embargo, en una red libre de escala la distribucin de grados sigue una ley de potencias. Por eso, al pintarlo en escala logartmica, aparece una recta.
